To run the tcp server, open terminal in the tptcp folder then run the command:
python .\tptcpserver.py

To run the tcp client, open terminal in the tptcp folder then run the command:
python .\tptcpclient.py

To run the udp server, open terminal in the tpudp folder then run the command:
python .\udpserver.py

To run the udp client, open terminal in the tpudp folder then run the command:
python .\udpclient.py
